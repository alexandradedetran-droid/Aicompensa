Cada card deve mostrar:

- Categoria

- Mercado

- Curtidas e validações

  - Curtir

  - Validar

5. Sistema de denúncia

Criar botão “Denunciar preço errado”.

Ao clicar:

- aumentar contador de denúncias

- se tiver 3 denúncias, marcar oferta como “suspeita”

- se passar da validade, marcar como “expirada”

6. Filtros avançados

Na tela de ofertas criar filtros:

- buscar produto

- categoria

- raio

- ordenar por:

  - menor preço

  - mais próximo

  - mais validado

  - mais recente

7. Perfil do usuário

Criar tela de perfil simples com:

- Nome: Usuário Demo

- Pontos

- Quantidade de ofertas publicadas

- Quantidade de validações recebidas

- Nível do usuário:

  - Bronze: 0 a 99 pontos

  - Prata: 100 a 299 pontos

  - Ouro: 300 a 999 pontos

  - Diamante: 1000+

8. Ranking melhorado

Mostrar ranking com:

- posição

- status

Quero visual de aplicativo real:
